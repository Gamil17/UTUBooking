# Email CTA Writer

## CTA Principles
- Specific beats generic: 'Search Makkah hotels' beats 'Click here'
- First-person converts: 'Find my hotel' beats 'Find your hotel'
- Verb-first always: start with an action word
- Button text: max 5 words
- Hyperlinked CTA line: 7-12 words

## For Each Request, Generate 3 Variants

Variant 1: **Benefit-led** — 'Get [specific result]'
Variant 2: **Action-led** — 'Search / Book / Compare / Find [specific thing]'
Variant 3: **Urgency/seasonal** — 'Book before Ramadan prices rise' / 'Secure your spot'

## Score Each Variant

| Variant | Clarity (1-5) | Specificity (1-5) | Urgency (1-5) | Total |
|---|---|---|---|---|
| 1 | | | | |
| 2 | | | | |
| 3 | | | | |

Clarity: Is it obvious what happens when you click?
Specificity: Does it name the exact thing they get?
Urgency: Is there a reason to click now rather than later?

**Booking CTAs:**

- 'Find my Umrah hotel'

- 'Book my Umrah package'

**Urgency CTAs:**
- 'Book before Ramadan rates apply'
- 'Secure Zone 1 before it sells out'

**Content/Awareness CTAs:**
- 'Read the full booking guide'
- 'See how the proximity filter works'
- 'Watch the 2-minute explainer'

## Context Rules
- Nurture emails: lower urgency, higher specificity
- Promotional emails: urgency-led, benefit-clear
- Transactional emails: action-led, zero filler
