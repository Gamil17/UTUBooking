# Content Repurposing Skill
Reference: skills/seo-content-writer/references/tone-of-voice.md
Anti-slop: run skills/anti-slop.md on all outputs before delivering

## Input
Paste or reference the source content (blog post URL, file path, or text).
State which formats to generate and which market to target.

## Outputs

### LinkedIn Post
Apply skills/marketing/linkedin-post-writer.md structure.
Pull the strongest insight or data point from the source as the hook.
Always end with a link or CTA back to the full post or UTUBooking page.

### Instagram Caption
Hook line (first sentence — visible before 'more' fold).
3-5 sentences of value — specific, not generic.
1 CTA sentence: 'Link in bio to book.' or 'Save this for your Umrah planning.'
8-12 relevant hashtags on a separate line:
  #Umrah #Hajj #MakkahHotels #IslamicTravel #UTUBooking #HajjPackages
  #UmrahPackages #MakkahHotel #HalalTravel + 3 market-specific tags

### WhatsApp Broadcast (Arabic — Gulf market)
Short. 3-4 sentences maximum. Gulf dialect preferred.
One clear action: "احجز الآن" or "اضغط للتفاصيل"
Flag with [REVIEW: native Arabic speaker] before sending.

### Email Snippet (100-150 words)
Paragraph format. Conversational. One main insight from the source.
Ends with a link to the full post with UTM tag:
`?utm_source=email&utm_medium=newsletter&utm_campaign=[slug]`

## Anti-Slop Check
Run all 4 outputs through skills/anti-slop.md before delivering.
Pay particular attention to Instagram and WhatsApp — brevity exposes filler fast.
