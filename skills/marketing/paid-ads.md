# Paid Ads Skill
Reference: skills/seo-content-writer/references/tone-of-voice.md
Anti-slop: run skills/anti-slop.md — ad copy is especially vulnerable to filler

## Inputs Required
Ask if not provided: campaign goal, market/country, budget tier, seasonal trigger

## Google Search Ads

**For each variant (generate 3):**
- Headline 1 (30 chars): Primary keyword + proximity or price signal
- Headline 2 (30 chars): Differentiator (walking time / halal filter / multi-currency)
- Headline 3 (30 chars): CTA ('Book Now' / 'Search Hotels' / 'Find Packages')
- Description 1 (90 chars): Specific benefit with number — distance, savings, or count
- Description 2 (90 chars): Trust signal + secondary CTA

**UTUBooking headline bank (adapt per campaign):**
- 'Hotels 150m From Haram Gate' (30 chars)
- 'Filter by Walking Time to Haram' (31 chars — trim to fit)
- 'Umrah Packages From [City]' (market-specific)
- 'Halal-Certified Hotels Verified' (31 chars)
- '200+ Makkah Hotels — Book Now'

## Meta Ads (Facebook + Instagram)

**For each variant (generate 3):**
- Primary text (125 chars visible): Hook. Problem. Tease of solution.
- Headline (40 chars): Specific benefit or seasonal urgency
- Description (30 chars): Supporting claim

**Three angle templates:**
1. Pain point: 'Walked 800m to Fajr because of a bad hotel choice...'
2. Aspiration: 'Your grandmother walked to Tawaf on her own. 150m from Gate 79.'
3. Social proof: '35 Malaysian pilgrims. One group booking. Zero issues.'

## Campaign Targeting Recommendations

Always include targeting recommendation per variant:
- Who to target (demographics, interests, behaviours)
- Negative audiences to exclude (e.g., exclude existing customers for acquisition campaigns)
- Winning prediction with rationale based on angle

## Rules
- No guaranteed / cheapest / best-in-class
- No competitor names in ad copy
- SAR pricing for KSA campaigns; GBP for UK; USD for US
- Seasonal urgency must be accurate — only use real deadlines
