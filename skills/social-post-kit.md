# Social Post Kit
Reference: skills/seo-content-writer/references/tone-of-voice.md
For deeper per-platform work: see skills/marketing/linkedin-post-writer.md
Anti-slop: run skills/anti-slop.md before delivering

## Inputs Required
- Content source (announcement, blog post URL, or brief)
- Market target (KSA / UK / US / MY / Global)
- Language (EN / AR / both)

## Platform Outputs

### LinkedIn
Apply skills/marketing/linkedin-post-writer.md structure:
Hook line → 3-5 value paragraphs → CTA
No hashtags. No em dashes. Max 1,300 characters.
Makkah/Madinah focus when relevant to Gulf audience.

### Instagram Caption
Hook line (visible before fold).
3-5 sentences of value — specific, not generic.
1 CTA: 'Link in bio to book.' / 'Save this for Umrah planning.'
8-12 hashtags on separate line:
  #Umrah #Hajj #MakkahHotels #IslamicTravel #UTUBooking
  + 3-5 market-specific or topic-specific tags

### WhatsApp Broadcast
Short — 3-4 sentences maximum.
Gulf Arabic dialect if Arabic requested.
Direct CTA: "احجز الآن" / "اضغط للتفاصيل" (or English equivalent).
Flag Arabic with [REVIEW: native Arabic speaker] before sending.

## Arabic Variants
When `AR` is requested:
- LinkedIn: provide full Arabic version with RTL note
- Instagram: Arabic caption replacing English (not appended)
- WhatsApp: Gulf dialect, verified by native speaker before sending
- Flag all Arabic output: [REVIEW: native Arabic speaker required]

## Output Order
Deliver in this sequence:
1. LinkedIn (EN)
2. Instagram (EN)
3. WhatsApp (EN or AR per market)
4. Arabic variants if requested (with [REVIEW] flags)
