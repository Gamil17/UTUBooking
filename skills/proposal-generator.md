# Skill: Proposal Generator
## Trigger: When new B2B hotel lead is qualified
## Steps:
1. Pull client info from HubSpot CRM via MCP
2. Identify hotel type: budget / mid / luxury / Haram-view
3. Generate proposal sections:
   - Executive Summary (EN + AR)
   - UTUBooking platform overview
   - Commission structure (SAR pricing)
   - Integration timeline (4 weeks)
   - Next steps + contact details
4. Save to Google Drive /Proposals/ folder
5. Notify CEO in Slack with review link
