Full REST API white-label — Careem builds their travel UI, UTUBooking provides hotel/flight/car inventory. Volume-tiered pricing. USD 84K/year API license (Growth tier, up to 5M API calls/month).

### The Exclusivity Clause — CEO Must Decide
We are offering Careem **24-month exclusivity in the ride-hailing-to-travel vertical across 8 GCC + regional markets** in exchange for a committed GMV of USD 8M/month by Month 12.

**Upside:** This is a major competitive moat — locks out Uber Travel from a white-label deal with us if/when they expand to GCC.

**Downside:** If Careem underperforms (< USD 8M GMV/month by M12), we're still bound for 24 months unless we invoke the break clause.
