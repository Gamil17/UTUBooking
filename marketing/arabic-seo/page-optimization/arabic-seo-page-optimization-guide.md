# دليل تحسين صفحات UTUBooking لبحث عربي (Arabic SEO On-Page Guide)

## القسم الأول: مبادئ البحث العربي

### كيف يختلف البحث العربي؟

| المعيار | الإنجليزية | العربية |
|--------|-----------|---------|
| اتجاه القراءة | LTR | RTL |
| أشكال الكلمة | محدودة | تصريفات وصيغ متعددة |
| البحث الصوتي | شائع | متزايد بسرعة |

## القسم الثاني: الكلمات المفتاحية المستهدفة لكل صفحة

### صفحة البحث عن الفنادق (`/hotels`)

**الكلمة الرئيسية:** `فندق قريب من الحرم المكي`

**كلمات ثانوية ذات حجم بحث عالٍ:**
```
- فنادق مكة المكرمة                    [بحث شهري: 40,500]
- حجز فندق مكة                          [27,100]
- فنادق قريبة من الكعبة                [12,000]
- أفضل فنادق مكة 2026                  [9,900]
- فندق قريب من الحرم رخيص              [8,100]
- فنادق المدينة المنورة               [18,000]
- حجز فندق المدينة المنورة             [14,800]
```

**تحسينات مطلوبة للصفحة:**

```html
<!-- العنوان المقترح (title tag) -->
<title>حجز فنادق قريبة من الحرم المكي والمدينة المنورة | UTUBooking</title>

<!-- الوصف (meta description) -->
<meta name="description"
  content="احجز فندقك في مكة المكرمة والمدينة المنورة بأفضل الأسعار.
  أكثر من 500 فندق موثّق بالقرب من الحرم. إلغاء مجاني. دفع بمدى وSTC Pay.">

<!-- عنوان H1 -->
<h1>فنادق قريبة من الحرم المكي — أفضل الأسعار والمواقع</h1>

<!-- Structured Data: Hotel -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Hotel",
  "name": "فنادق مكة المكرمة",
  "description": "أفضل فنادق بالقرب من المسجد الحرام",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "مكة المكرمة",
    "addressCountry": "SA"
  }
}
</script>
```

## القسم الثالث: تحسين المحتوى العربي (Content Optimization)

### 3.1 — بنية URL العربية

**التوصية:** استخدم URLs إنجليزية + slugs وصفية
```
✅ utubooking.com/hotels/makkah-near-haram
✅ utubooking.com/flights/jeddah-dubai
✅ utubooking.com/cars/jeddah

❌ utubooking.com/hotels/فنادق-مكة  (قد يتسبب في مشاكل encoding)
```

### 3.2 — عناوين H2 وH3 موجّهة للبحث العربي

**قاعدة:** لكل صفحة، اكتب على الأقل:
- 1 عنوان H1 يحتوي الكلمة المفتاحية الرئيسية
- 3-5 عناوين H2 تحتوي كلمات ثانوية
- نص أصيل (لا نسخ) لكل قسم

**مثال للصفحة الرئيسية للفنادق:**
```
H1: حجز فنادق في مكة المكرمة والمدينة المنورة
  H2: أفضل فنادق قريبة من الحرم المكي
    H3: فنادق 5 نجوم بجوار الكعبة
    H3: فنادق 4 نجوم بسعر مناسب
  H2: فنادق المدينة المنورة قريبة من المسجد النبوي
  H2: نصائح لاختيار الفندق في موسم الحج
  H2: أسئلة متكررة عن فنادق مكة
```

### 3.3 — الأسئلة الشائعة (FAQ Schema)

أضف قسم FAQ لكل صفحة رئيسية مع البيانات المنظّمة:

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "ما هو أقرب فندق من الحرم المكي؟",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "أقرب الفنادق من الحرم المكي هو فندق فيرمونت مكة (أبراج البيت) على بُعد 50 متر فقط من البوابة الرئيسية."
      }
    },
    {
      "@type": "Question",
      "name": "كيف أحجز فندقًا في مكة المكرمة؟",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "يمكنك الحجز مباشرة عبر UTUBooking باختيار مكة المكرمة كوجهة وتحديد تواريخ الإقامة وعدد الضيوف."
      }
    }
  ]
}
</script>
```

## القسم الرابع: التحسين التقني (Technical SEO)

### 4.1 — إعدادات اللغة والاتجاه

```html
<!-- في كل صفحة عربية -->
<html lang="ar" dir="rtl">

<!-- hreflang للصفحات الثنائية اللغة -->
<link rel="alternate" hreflang="ar" href="https://utubooking.com/ar/hotels/makkah">
<link rel="alternate" hreflang="en" href="https://utubooking.com/en/hotels/makkah">
<link rel="alternate" hreflang="x-default" href="https://utubooking.com/hotels/makkah">
```

### 4.2 — سرعة الصفحة (Core Web Vitals)

أهداف مقاييس الأداء للسوق السعودي (شبكات 4G-5G):
```
LCP (Largest Contentful Paint): < 2.5 ثانية
FID (First Input Delay): < 100ms
CLS (Cumulative Layout Shift): < 0.1
```

**أولويات التحسين:**
- ضغط الصور بصيغة WebP
- Lazy loading للصور خارج الشاشة
- تخزين مؤقت (CDN) في منطقة الخليج (AWS Bahrain)

### 4.3 — البحث الصوتي العربي

الاستعلامات الصوتية الأكثر شيوعًا:
```
"أريد أحجز فندق في مكة"
"كم سعر السيارة في جدة"
"وين أقرب فندق من الحرم"
"كيف أحجز رحلة رخيصة للكويت"
```

**التوصية:** أضف محتوى يُجيب على هذه الأسئلة بلغة طبيعية في قسم FAQ.

## القسم الخامس: بناء الروابط الخلفية (Backlinks) العربية

### مواقع عربية مستهدفة للروابط:
1. **منتديات ملتقى الحجاج** — مشاركة تجارب وتوصيات الفنادق
2. **مدونة وزارة السياحة السعودية** — محتوى مفيد + ذكر UTUBooking
3. **عين السفر** — مراجعات سفر عربية
4. **مواقع الشيوخ والدعاة** — شراكات لمحتوى رحلات العمرة والحج
5. **LinkedIn العربي** — مقالات B2B لشركات السفر والسياحة
